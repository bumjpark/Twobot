#프로젝트 소개
Twobot은  두 개의 AI 챗봇인 Gemini와 ChatGPT를 동시에 활용하여 사용자가 두 가지의 챗봇에서 답장을 확인할 수 있는 안드로이드 애플리케이션입니다.
이 앱은 1:1 채팅과 토론 기능을 제공하며 사용자가 기능을 더 잘 사용 할 수 있게 만들었습니다.
